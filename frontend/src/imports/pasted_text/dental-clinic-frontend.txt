Generate Dental Clinic Management System Frontend

You are a senior frontend engineer and UI/UX designer. Build a complete modern frontend application for a Dental Clinic Management System.

The application is used by dentists to manage patients, appointments, dental records, files, visits, and billing.

The frontend must be production-ready, clean, responsive, and designed for a medical environment.

Technology Requirements

Build using:

React + TypeScript
Vite
Tailwind CSS
React Router
Axios for API communication
React Query / TanStack Query for server state management
A component library such as shadcn/ui or equivalent
Form handling with React Hook Form + Zod validation
Modern responsive dashboard layout

Authentication

Implement authentication flow:

Login page:

POST /api/auth/login

Request:

{
 "email":"",
 "password":""
}

Response:

{
 "access_token":"",
 "user":{
   "id":"",
   "role":"doctor",
   "full_name":""
 }
}

Requirements:

Save JWT token securely
Axios interceptor automatically adds:
Authorization: Bearer TOKEN
On application startup:
GET /api/auth/me

Restore user session.

Handle:

401:
 redirect to login

403:
 user authenticated but no permission
 disable create/edit/delete actions

Create centralized permission system:

Example:

canCreate()
canEdit()
canDelete()

Do not create admin panels or role switching.
